Technology

TEI

Heeding Peter Robinson's admonition that "your interface is everyone else's enemy", we decided to expose the TEI version of the papers in two ways:

  • Bulk Download: A Github repository containing the editor's transcripts, the TEI-P5 XML files, and the programs used to convert the transcripts to TEI-XML is at AustinTranscripts and can be downloaded for analysis or any other re-use.
  • On-Screen TEI: Each page on the Digital Austin Papers site has a TEI button: . Clicking this will download that document's TEI-XML, formatted using TEI Boilerplate.
